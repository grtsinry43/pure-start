一个现代化，美观简洁高效的浏览器起始页，使用 Next.js 构建

---

## 特性

- 美观简洁的 UI 设计，支持暗黑模式
- 使用 Next.js 构建，支持服务端渲染
- 使用 TypeScript 编写，类型安全
- 使用 Shadcn-ui 组件库，支持主题定制

## 参与开发

```bash
# 克隆项目
git clone https://github.com/grtsinry43/pure-start.git

# 进入项目目录
cd pure-start

# 安装依赖
pnpm install

# 启动项目
pnpm dev
```
